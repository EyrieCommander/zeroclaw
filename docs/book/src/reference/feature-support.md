# Feature support matrix

This page compares ZeroClaw support against nearby Claw-family runtimes and deployment shapes. It is meant to answer "which claw supports this?" quickly, then point readers to the canonical docs for details.

The matrix is conservative. Hover a status symbol for the full meaning. A row marked <abbr title="Needs verification: this project needs a current source/docs pass before making a support claim">?</abbr> is not a current support claim.
